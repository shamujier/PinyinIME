# PinyinIME

#### 介绍
基于googlepinyin输入法改造

#### 软件架构
软件架构说明
AndroidStudio3.4.1

Gradle plugin3.4.1

Gradle 5.1.1

NDK 19.2.5345600 & CMake 3.6.4111459

ANDROID_HOME 安卓SDK系统环境变量根基自己定义修改，在app/build.gradle
layoutlib.jar 采用的是android-22 在app/build.gradle根基系统自己修改
